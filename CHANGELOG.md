## [1.0.0] - 2025-12-28

### Added

- Initial release of Orders module
- **Core Features**
  - Order management
  - Station routing
  - KDS integration
  - Order status tracking
  - Ticket printing

- **Models**
  - `Order`: Core order entity
  - `OrderItem`: Order line items
  - `Station`: Kitchen/bar stations
  - `RoutingRule`: Order routing configuration

- **Internationalization**
  - English translations (base)
  - Spanish translations
